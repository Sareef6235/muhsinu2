üöÄ How to Use This Template

  1. Copy this HTML structure to your pages
  2. Update the data-page-id attribute on <body>
  3. Add your content in the <main> section
  4. Customize navigation and footer in config files

üìù Integration Steps

1. Include CSS Files

Add these in your <head> section:

<link rel="stylesheet" href="assets/css/global-navigation.css">
<link rel="stylesheet" href="assets/css/global-footer.css">

2. Add Container Divs

Add these in your <body>:

<body data-page-id="your-page-id">
  <div id="global-nav"></div>
  
  <main id="main-content" style="padding-top: 90px;">
    <!-- Your content here -->
  </main>
  
  <div id="global-footer"></div>
</body>

3. Include Site Editor

Add this before closing </body> tag:

<script src="assets/js/site-editor.js"></script>
<script>
  window.addEventListener('DOMContentLoaded', () => SiteEditor.init());
</script>
